## 🇪🇸 Descripción en Español

**MultiPass** es una solución de vanguardia para la verificación de credenciales de correo electrónico. Desarrollada con **WPF** y un diseño enfocado en la experiencia de usuario (UX), esta herramienta permite procesar grandes volúmenes de datos con una precisión excepcional y un consumo de recursos optimizado.

### 🚀 Características Principales
- **Motor Multi-hilo:** Ejecución ultra-rápida capaz de manejar miles de solicitudes por minuto.
- **Soporte de Proxies:** Integración avanzada de proxies para evitar detecciones y bloqueos.
- **Estadísticas en Tiempo Real:** Panel de control detallado con tasas de éxito, velocidad y errores.
- **Interfaz Dark Modern:** Estética profesional basada en tonos oscuros con visualizaciones fluidas.
- **Búsqueda Dinámica:** Filtrado instantáneo de resultados por estado o texto.

### 🛠️ Cómo Utilizar
1. **Descarga:** Obtenga la última versión desde la sección de [Releases](https://github.com/KONDORDEVSECURITYCORP/MultiPassWPF/releases).
2. **Carga de Datos:** Utilice el botón \Load Credentials\ para importar su lista \user:pass\.
3. **Configuración:** Configure sus proxies en \Setup Proxies\ si requiere anonimato.
4. **Ejecución:** Presione \Execute\. Puede pausar o reanudar el proceso en cualquier momento.
5. **Análisis:** Revise los resultados en la tabla central y exporte los datos necesarios.

## 🇺🇸 English Description

**MultiPass** is a cutting-edge solution for email credential verification. Developed with **WPF** and a UX-focused design, this tool allows processing large volumes of data with exceptional precision and optimized resource consumption.

### 🚀 Main Features
- **Multi-threaded Engine:** Ultra-fast execution capable of handling thousands of requests per minute.
- **Proxy Support:** Advanced proxy integration to prevent detection and blocks.
- **Real-time Statistics:** Detailed dashboard with success rates, speed, and errors.
- **Dark Modern Interface:** Professional aesthetics based on dark tones with fluid visualizations.
- **Dynamic Search:** Instant filtering of results by status or text.

### 🛠️ How to Use
1. **Download:** Get the latest version from the [Releases](https://github.com/KONDORDEVSECURITYCORP/MultiPassWPF/releases) section.
2. **Data Loading:** Use the \Load Credentials\ button to import your \user:pass\ list.
3. **Configuration:** Set up your proxies in \Setup Proxies\ if anonymity is required.
4. **Execution:** Press \Execute\. You can pause or resume the process at any time.
5. **Analysis:** Review the results in the center table and export the necessary data.

## 📋 Requisitos / Requirements
- **OS:** Windows 10/11 (64-bit)
